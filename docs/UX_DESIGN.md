# UX/UI Design Concept - Easel

## 1. Visual Identity
- **Concept:** "Warmth & Understanding"
- **Color Palette:**
  - **Peach (#FFDAB9):** Primary action color, cycle tracking.
  - **Cream (#FFFDD0):** Background, soft and non-intrusive.
  - **Sage Green (#B2AC88):** Health, calm, follicular phase.
  - **Deep Mauve (#673147):** SOS button, high priority.
- **Typography:** 
  - Serif (Header): "New York" (Apple System Font) for elegance.
  - Sans-Serif (Body): "SF Pro" for clarity.

## 2. Core Screens

### 2.1. The Dashboard (Girlfriend)
- **Phase Wheel:** A pulsating, soft-edged circle showing the current day in the cycle.
- **Status Label:** "Phase: Follicular. You're glowing!"
- **Quick Log:** Horizontal scroll of icons for mood (Happy, Sad, Sensitive, Angry).
- **The "Easel SOS" Button:** A floating heart icon at the bottom right.

### 2.2. The Dashboard (Boyfriend)
- **Her Status View:** High-level view of where she is in her cycle.
- **Daily Advice Card:** A dynamic text box that changes based on her phase and logged symptoms.
- **Suggested Actions:** "Buy Flowers", "Make Tea", "Order Comfort Food".

### 2.3. The SOS Menu (The Nudge)
- **Overlay UI:** Blurs the background, pops up a radial menu.
- **Options:** 
  - 🍫 Sweet Tooth
  - 🧸 Need Physical Comfort
  - 💊 Pain Relief
  - 🌙 Quiet Time
- **Animation:** Lottie-based heart animation when sent.

### 2.4. Calendar View
- **Multi-month scroll:** Standard calendar with subtle color coding.
- **Event Markers:** "Estimated Period", "Estimated Ovulation".
- **Shared Notes:** Add a note for both to see (e.g., "Beach Trip Plan").

## 3. Interaction Design
- **Haptics:** Soft haptic feedback when logging or sending an SOS.
- **Transitions:** Smooth fade and slide transitions between screens to maintain a "calm" app environment.
