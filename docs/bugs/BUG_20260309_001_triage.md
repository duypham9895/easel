## Bug Group A — Critical Logic Bug

### BUG-A: Cycle calculator wraps via modulo (ISS-001)
- **Severity:** Critical
- **File:** `app/utils/cycleCalculator.ts:19`
- **Steps:** Set lastPeriodStartDate 33 days ago with 28-day cycle → app shows Day 5 (phantom menstrual phase) instead of Day 33 (late period)
- **Expected:** Show Day 33, indicate period is late
- **Actual:** `(33 % 28) + 1 = 6` — shows Day 6, wraps to follicular phase
- **Root cause:** `return (diffDays % avgCycleLength) + 1` — modulo wraps unconditionally
- **Frequency:** Always, for any user whose period is late
- **Affected scope:** All Moon and Sun users when period exceeds expected cycle length
- **Related features:** MoonDashboard, SunDashboard, Calendar, all phase-dependent features
