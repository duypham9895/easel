## BUG-A: Cycle Calculator Modulo Fix (CRITICAL)

**File:** `app/utils/cycleCalculator.ts`

**Change:** Removed modulo wrapping from `getCurrentDayInCycle()`. The function now returns the raw day count (diffDays + 1) without wrapping via `% avgCycleLength`. This means late periods show the actual day (e.g., Day 33 for a 5-day-late period in a 28-day cycle) instead of wrapping to a phantom phase.

**Before:**
```typescript
export function getCurrentDayInCycle(
  lastPeriodStartDate: string,
  avgCycleLength: number,
): number {
  // ...
  return (diffDays % avgCycleLength) + 1;
}
```

**After:**
```typescript
export function getCurrentDayInCycle(
  lastPeriodStartDate: string,
  _avgCycleLength: number,  // kept for API stability
): number {
  // ...
  if (diffDays < 0) return 1;
  return diffDays + 1;
}
```

**Test update:** `app/__tests__/cycleCalculator.test.ts` â€” renamed "wraps around after cycle completes" to "continues counting past cycle length for late periods", changed expected value from `3` to `31`. Also updated "handles 21-day short cycle" to expect `22` instead of `1`.

**Impact:** All cycle-dependent features (MoonDashboard, SunDashboard, Calendar, phase wheel) now correctly show late periods instead of phantom menstrual phases.

## BUG-C: Tab Bar Role-Aware Theming (HIGH)

**File:** `app/app/(tabs)/_layout.tsx`

**Change:** Added role-aware color computation for tab bar background, active tint, and inactive tint. Previously hardcoded to `Colors.card` (white) and `Colors.menstrual` (pink).

**Added imports:**
- `MoonColors, SunColors` from `@/constants/theme`
- `useAppStore` from `@/store/appStore`

**Logic:**
```typescript
const role = useAppStore(s => s.role);
const tabBg = role === 'moon' ? MoonColors.card : role === 'sun' ? SunColors.card : Colors.card;
const activeTint = role === 'moon' ? MoonColors.accentPrimary : role === 'sun' ? SunColors.accentPrimary : Colors.menstrual;
const inactiveTint = role === 'moon' ? MoonColors.textHint : role === 'sun' ? SunColors.textHint : Colors.textHint;
```

**Result:** Moon sees dark indigo tab bar with lavender active tint. Sun sees cream tab bar with amber active tint. Null role falls back to default white/pink.
