# BUG_20260308_002 — Notify Cycle Daily CI Pipeline Fails (HTTP 404)

## Bug Report

**Bug ID:** BUG_20260308_002
**Title:** Notify Cycle Daily GitHub Actions workflow fails — edge function not found on Supabase
**Severity:** High — core feature broken (daily cycle notifications never sent), no workaround
**Frequency:** Always (every scheduled run fails)
**Affected scope:** All users — no Moon or Sun users receive daily cycle push notifications

## Environment

- **Platform:** GitHub Actions (ubuntu-latest) → Supabase Edge Functions
- **Workflow:** `.github/workflows/notify-cycle.yml`
- **Schedule:** Daily at 8:00 AM UTC
- **App version:** v1.5.1

## Steps to Reproduce

1. GitHub Actions cron triggers `notify-cycle.yml` at 8:00 AM UTC
2. Workflow sends `POST https://<project>.supabase.co/functions/v1/notify-cycle` with service role key
3. Supabase returns HTTP 404: `{"code":"NOT_FOUND","message":"Requested function was not found"}`
4. Workflow exits with code 1

## Expected Behavior

- Supabase returns HTTP 200
- Edge function processes Moon users and sends push notifications for period approaching/started/ended

## Actual Behavior

- HTTP 404 — `notify-cycle` edge function does not exist on Supabase
- No notifications sent
- Workflow fails every day (confirmed on 2026-03-07 and 2026-03-08)

## Root Cause Hypothesis

The `notify-cycle` edge function code exists at `app/supabase/functions/notify-cycle/index.ts` but was **never deployed** to Supabase production. The deployment pipeline (`supabase.yml` → `deploy-functions` job) only runs when commits modify `app/supabase/functions/` files. Since recent commits only touched docs and UI files, the `Edge Functions` job is always **skipped**.

Evidence:
- Latest Supabase workflow run (22819129010): `Edge Functions` job = **skipped**
- `notify-cycle.yml` run 22817076829: HTTP 404
- `notify-cycle.yml` run 22795305397 (day before): HTTP 404
- Git history: function files last modified in commit `2eb1eea` (early batch commit)

## Related Features Potentially Affected

- `notify-sos` edge function (same deployment condition — may also be undeployed)
- All Moon/Sun push notifications (cycle approaching, started, ended)
