## 5. Emotional Safety Boundaries

### Topics to Monitor
- **Surveillance dynamics** — Does the app make Linh feel watched by Minh?
- **Data weaponization** — Could Minh use cycle data to dismiss Linh's emotions?
- **Body judgment** — Does any copy make Linh feel abnormal or "hormonal" in a reductive way?
- **Pressure to engage** — Does the app guilt-trip either user for not checking in?
- **Intimacy boundaries** — Do whisper/SOS features feel supportive or invasive?
- **Fertility anxiety** — Does any feature trigger unwanted fertility/pregnancy concerns?

### Red Flags to Watch For
- App language that implies "your hormones are making you act this way"
- Any feature that reveals engagement metrics to the partner
- Notifications that create obligation rather than invitation
- Copy that treats the cycle as a problem rather than a natural process
